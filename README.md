# Music Explorer

Music Explorer is a song recommendation application/tool. Input a song, and the AI will suggest a list of similar tracks based on a dataset.

You can find the download link for the .exe file over in the releases section.

Here is a video presentation of the idea and workings of the application as well as a demo: [Video](https://youtu.be/PRlVQ9Ndxyo)

---

Key Features:
* **Genre Filtering:** Narrow down your results by excluding or including certain genres.
* **Random Song:** Get a surprise recommendation with one click.
* **Search History:** Quickly access your five recently searched tracks.
* **Direct Play:** Redirects you to the music on YouTube with an integrated play button.

# Note on Data
The application's recommendations are limited to the specific dataset provided. Not every song in the world is available for search.

*When downloading the source code, make sure to place the dataset.csv, app.py, and recommender.py all in one folder, and open the folder with VSCode or any other code editor application that you use to ensure that the python scripts are able to read the dataset.csv file.*
